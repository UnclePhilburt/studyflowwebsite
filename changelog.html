Changelog

Latest updates and improvements to StudyFlowSuite

v0.0.4 Latest March 19, 2026

Fixed

  • Fixed "Invalid launch" errors in Canvas sequential mode by increasing delays between answer clicks and submit button
  • Improved Canvas LTI session handling to prevent 403 errors during page transitions
  • Extended wait time to 3 seconds after clicking answer and 4 seconds after clicking submit

Improved

  • More reliable quiz navigation in Canvas sequential mode
  • Better compatibility with Canvas LTI authentication flow
v0.0.3 March 19, 2026

New Features

  • Added support for multiple-answer multiple choice questions (checkboxes)
  • Extension now detects checkbox vs radio button questions automatically
  • AI returns multiple correct answers for "select all that apply" questions

Improved

  • Backend API now handles both single-answer and multiple-answer question types
  • Click logic loops through all correct checkboxes with 500ms delay between clicks
  • Answer display shows all selected options for multiple-answer questions
v0.0.2 March 19, 2026

Improved

  • Removed unused scripting permission (Chrome Web Store requirement)
  • Removed Dev Inspector feature for production release
  • Simplified host permissions to backend API only
  • Uses activeTab permission for better security and user control

Fixed

  • Extension now works on any school website domain
  • Improved compatibility with custom Canvas installations
v0.0.1 Initial Beta March 2026

New Features

  • Initial release of StudyFlowSuite Chrome extension
  • Support for Canvas LMS and Learnosity quiz platforms
  • Sequential mode: One question per page with auto-navigation
  • Single-page mode: All questions visible on one page
  • Multi-question mode: Multiple questions per page with manual navigation
  • AI-powered answer detection using OpenAI GPT models
  • Support for multiple choice, essay, and fill-in-blank questions
  • User authentication and subscription management
  • Free tier: 10 questions per day
  • Pro tier: Unlimited questions with advanced models
