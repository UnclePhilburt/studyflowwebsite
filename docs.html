Quick Start

First Time Setup

  1. Sign into your StudyFlowSuite account
  2. Navigate to the chat assistant
  3. StudyFlowSuite will guide you through a 3-step setup wizard
  4. Configure your quiz settings (optional)
  5. Click "Start Quiz"
Pro Tip: Enable "One-page quiz mode" if all questions appear on a single page instead of clicking through pages.

Setup Wizard

The first time you use StudyFlowSuite on a new website, you'll be guided through a simple 3-step setup.

Step 1: Select Question Element

Click on the question text on your quiz page. This helps StudyFlowSuite identify where questions are located.

Step 2: Select Answer Area

Click on any answer option. This teaches StudyFlowSuite where answer choices appear.

Step 3: Select Submit Button

Click on the Submit or Next button. This allows automatic progression through the quiz.

Important: These settings are saved per-domain. You only need to set up once per quiz platform!

Skipping Setup

If you enable One-page quiz mode, the setup wizard is automatically skipped since StudyFlowSuite uses different detection methods.

Supported Question Types

Multiple Choice

The most common question type. StudyFlowSuite analyzes all answer options and selects the correct one.

  • Single selection (radio buttons)
  • Multiple selection (checkboxes)
  • True/False questions

Fill in the Blank

Short answer questions where you need to provide a specific word or phrase.

Example: The capital of France is _______.

Short Answer

Questions requiring a brief text response (2-5 sentences).

Essay Questions

Long-form questions requiring detailed, multi-paragraph responses. StudyFlowSuite generates well-structured, contextually appropriate essays.

Quiz Modes

Sequential Mode (Default)

For quizzes where questions appear one at a time:

  • Answer one question
  • Click Submit/Next automatically
  • Move to next question
  • Repeat until quiz complete

One-Page Mode

For quizzes where all questions are visible at once:

  • Scans all visible questions
  • Answers questions in order
  • Pauses when page is complete
  • Prompts user to navigate manually
How to choose: Check the "One-page quiz mode" checkbox if you can scroll and see multiple questions at once.

Smart Pacing

Make your quiz completion look natural and human-like.

How It Works

Smart Pacing distributes your quiz time naturally across all questions:

  1. Set your target completion time (in minutes)
  2. Set total number of questions
  3. StudyFlowSuite calculates optimal delays between questions
  4. Adds random variation (±30%) to appear natural

Example

Total Questions: 50
Target Time: 25 minutes

Result: ~30 seconds per question
         (with 20-40 second variation)
Note: Leave both fields empty for instant completion with no delays.

Settings

Total Questions

Set the expected number of questions in your quiz. Used for:

  • Smart pacing calculations
  • Auto-stop when limit reached
  • Progress tracking

Target Time

How long you want the quiz to take (in minutes). Requires Total Questions to be set.

One-Page Quiz Mode

Toggle between sequential and one-page detection modes.

Clear Saved Setup

Reset the setup wizard for the current website. Use this if:

  • The quiz layout has changed
  • Detection is not working correctly
  • You want to reconfigure element selectors

Common Issues

Not Detecting Questions

Solution:

  1. Refresh the page
  2. Clear saved setup for this site
  3. Run setup wizard again
  4. Try toggling "One-page quiz mode"

Wrong Answers Being Selected

Solution:

  1. Check that question text is being captured correctly (view console logs)
  2. Ensure all answer options are visible
  3. Try refreshing and restarting

Stuck After Completing Page

Solution:

  • This is expected behavior! Click the displayed Next/Submit button, then click "Resume"
  • StudyFlowSuite pauses to let you control page navigation

Essay Answers Are Generic

Solution:

  • Make sure the full question text is being captured
  • Check console logs to see what text is being processed
  • Question might need more context for better responses

Frequently Asked Questions

Can teachers detect if I'm using this?

StudyFlowSuite operates like a human user - clicking buttons and filling forms. There's no definitive way for instructors to detect automated quiz taking vs. manual completion. However, always use responsibly and in accordance with your institution's academic policies.

Does it work on timed quizzes?

Yes! Use the Smart Pacing feature to ensure completion within your time limit. Set "Total Questions" and "Target Time" to automatically distribute time appropriately.

What happens if a question is answered incorrectly?

StudyFlowSuite tracks errors automatically. Some questions may be ambiguous or require specific course knowledge not available in the question text.

Can I pause mid-quiz?

Yes! Click the "Stop Quiz" button at any time. Your progress is saved. You can restart by clicking "Start Quiz" again.

Do I need an account?

No account needed for the free tier. Pro features require a subscription account.
