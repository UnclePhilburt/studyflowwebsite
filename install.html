Install StudyFlow Extension

Follow these simple steps to get started

📥 Download Extension (Latest Version)
1

Download the Extension

Click the download button above to get the latest version of the StudyFlow extension ZIP file.

2

Extract the ZIP File

Right-click the downloaded ZIP file and select "Extract All" (Windows) or double-click to extract (Mac). This will create a folder containing the StudyFlow-Extension folder.

3

Open Chrome Extensions

Open Google Chrome and go to chrome://extensions/ or click the three dots menu → Extensions → Manage Extensions.

4

Enable Developer Mode

In the top-right corner of the Extensions page, toggle on "Developer mode".

5

Load the Extension

Click "Load unpacked" button. Navigate to where you extracted the files and select the StudyFlow-Extension folder (the inner folder, not the outer one). Click "Select Folder".

6

Pin the Extension

Click the puzzle piece icon in Chrome's toolbar, find StudyFlow, and click the pin icon to keep it visible.

7

Log In & Start

Click the StudyFlow extension icon, log in with your account, and you're ready to go!

Note: This extension is currently in development and requires manual installation. Once published to the Chrome Web Store, installation will be much simpler!
