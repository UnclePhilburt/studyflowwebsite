Interactive Installation Assistant

Follow along as we guide you through each step

1
Download Extension

First, download the StudyFlow extension ZIP file to your computer.

📥 Download Extension

✓ Download started!

File saved to your Downloads folder

Next: Extract the ZIP file by right-clicking it

2
Extract ZIP File

Right-click the downloaded ZIP file and select "Extract All" (Windows) or double-click (Mac).

Important: Remember where you extract it! You'll need to navigate to this folder.

📁 Example: C:\Users\YourName\Downloads\StudyFlow-Extension
Need help?
Windows: Right-click → "Extract All" → Choose location → Click "Extract"
Mac: Just double-click the ZIP file
3
Open Chrome Extensions Page

Open Chrome's extensions management page.

chrome://extensions/ 📋 Click to copy

1. Copy the URL above (click it)
2. Open a new Chrome tab
3. Paste it in the address bar and press Enter

Alternative: In Chrome, click the three dots menu → Extensions → Manage Extensions
4
Enable Developer Mode

On the Extensions page, find the "Developer mode" toggle and turn it ON.

Look for: A toggle switch that says "Developer mode"
Location: Usually in the top-right corner, but may appear on the top-left for some users
When enabled: The toggle should turn blue/purple
5
Load Extension

Click "Load unpacked" button (appears after enabling Developer mode).

Then: Navigate to where you extracted the ZIP and select the StudyFlow-Extension folder (you need to open it, not just select the parent folder).

⚠️ Common mistake: The folder will appear empty in the dialog - that's OK! Just make sure you're INSIDE the StudyFlow-Extension folder and click "Select Folder"
Correct path: .../StudyFlow-Extension/
The folder should contain: manifest.json, popup.html, etc.
6
Verifying Installation

Let's check if the extension is properly installed...

🔍 Checking for StudyFlow extension...
Extension not detected?
• Make sure you selected the correct folder (StudyFlow-Extension)
• Check that manifest.json is in the folder you selected
• Try reloading: chrome://extensions → Click the refresh icon on StudyFlow card
• Then click "Check Again" above

🎉

Installation Complete!

StudyFlow extension is ready to use

Go to Dashboard

Don't forget to pin the extension to your toolbar for easy access!
