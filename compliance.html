Missouri AI Regulatory Compliance Compliant
GUARD Act (SB 1455) -- Age Verification
All users must be 18 or older. Age verified at registration via self-certification. Phase 2 (government-grade ID verification via Veratad API) scheduled before August 28, 2026 deadline.
Transparency and Accountability Act (SB 1324)
All AI-generated content is marked with provenance headers. Steganographic watermarking (zero-width characters) embedded in AI responses. Clipboard provenance appended on copy. Every AI interaction logged with 7-year encrypted retention.
AI Disclosure
Mandatory disclosure statement on every AI interaction: "You are chatting with StudyFlow AI, an artificial intelligence assistant. I am not a human." Rolling 30-minute interval reminders during active sessions.
DMCA Compliance
Full Digital Millennium Copyright Act compliance with takedown request processing, counter-notification support, and repeat infringer policies.
Data Security Active
Encryption in Transit
All data transmitted over HTTPS/TLS. No unencrypted connections accepted.
Authentication
Supabase Auth with bcrypt password hashing, JWT session tokens, and .edu email verification for academic features.
Row-Level Security
PostgreSQL Row-Level Security (RLS) enforced on all user data tables. Users can only access their own data. Study Group data scoped to group membership.
Document Server
ONLYOFFICE Document Server self-hosted on company infrastructure (DigitalOcean). Student documents are not processed by third-party SaaS providers.
Cyber Liability Insurance
Chubb cyber liability policy (A++ rated, AM Best) covering data breach response, regulatory defense, business interruption, and third-party liability.
Data Handling and Retention
AI Provenance Logging
Every AI interaction logged with user ID, conversation ID, prompt text, response text, sources used, model used, response time, and IP address. Retained for 7 years per Missouri SB 1324.
Upload Audit Trail
Every file upload logged with SHA-256 hash, timestamp, file size, IP address, and sharing preference for tamper verification.
Student Data Isolation
Personal notes are private by default. Nexus sharing is opt-in per upload. Study Group notes are scoped to group membership. No student data is sold or shared with advertisers.
Data Deletion
Users can request deletion of their account and all associated data at any time. Conversations are soft-deleted (retained for compliance) but hidden from the user interface.
