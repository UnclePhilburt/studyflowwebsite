📑 Tutorial: Multi-Page Quiz Format
This quiz has MULTIPLE questions per page across 3 pages - tests pause/resume
Mode: Multi-Page
💡 How to Use This Tutorial
1
Install Extension
Make sure StudyFlowSuite is installed
2
Click Extension Icon
Open the extension popup from toolbar
3
Keep Default Settings
Don't check "One-page quiz mode" for this quiz
4
Click "Start Quiz"
It will answer all 3 questions on page 1, then stop
5
Click "Next Page"
Manually navigate to page 2 after questions are answered
6
Click "Resume Quiz"
Extension continues with new questions on page 2
💡 How This Works:

The extension will answer all 3 questions on page 1, then STOP. You manually click "Next Page" to go to page 2. Once page 2 loads, click "Resume Quiz" in the extension to continue. It tracks which questions were already answered and only answers new questions. Repeat for page 3.
