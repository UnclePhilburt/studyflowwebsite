Smart Usage Guidelines

Get the most out of StudyFlow while staying safe and responsible

🛡️

Browser Extension Technology

StudyFlow runs as a standard Chrome extension using the same technology as ad blockers, password managers, and productivity tools used by millions daily. It operates entirely within your browser and never modifies your computer's system files.

🔒

Your Data Privacy

StudyFlow only processes quiz questions you explicitly activate it on. No browsing history, personal files, or unrelated data is collected. Question data is sent securely via HTTPS to our AI service for processing, then immediately discarded.

🎯

Learning Aid Tool

StudyFlow is designed as a study assistance tool. It works best for practice quizzes, self-assessments, and homework where the goal is to learn material efficiently. We recommend using it responsibly to supplement your education.

Platform Compatibility

✅ Works Great With:

  • Standard Canvas and other LMS platforms
  • Colibri Real Estate courses and exams
  • Basic online quizzes without proctoring
  • Practice tests and self-assessments
  • Homework assignments
  • Study guides and review materials

Most college websites don't actively monitor for browser extensions unless using specialized proctoring software.

⚠️ Not Recommended With:

  • Respondus LockDown Browser
  • Proctorio or Honorlock proctoring
  • ProctorU live proctoring
  • Examity or similar services
  • High-stakes exams with monitoring

Proctoring software specifically monitors for automation tools and browser extensions. StudyFlow will not work with these services and attempting to use it may violate academic integrity policies.

Technical Detection Information

Extension Detection:

Websites can detect browser extensions through various methods. StudyFlow operates like any standard extension (similar to Grammarly or LastPass) and is visible to sites that actively check for extensions.

Activity Patterns:

The extension automates form filling and clicking. On platforms with basic LMS (no proctoring), this activity appears similar to a fast typist. Advanced proctoring tools may flag automated patterns.

Network Requests:

StudyFlow communicates with our secure API to process questions. This creates network traffic that advanced monitoring could detect, though most standard LMS platforms don't monitor external API calls.

📚

Use Responsibly

StudyFlow is a powerful learning tool designed to help students work through practice materials efficiently. It's your responsibility to use it in accordance with your institution's academic integrity policies. We recommend using StudyFlow for:

  • Practice quizzes where the goal is learning, not assessment
  • Homework assignments that allow outside resources
  • Self-study and review materials
  • Time-saving on repetitive coursework

Important: Do not use StudyFlow on proctored exams, high-stakes assessments, or any scenario where its use would violate academic integrity policies. Always check your institution's policies before use.

Can websites detect StudyFlow?

Standard LMS platforms like Canvas without proctoring software typically don't monitor for extensions. However, advanced proctoring tools like Respondus, Proctorio, or Honorlock can detect browser extensions. See our safety section above for details.

Is it safe to use on my college quizzes?

StudyFlow works best with unproctored practice quizzes and homework. For high-stakes proctored exams, we strongly recommend NOT using it as proctoring software will detect it. Always check your institution's academic integrity policies.

Is my data safe?

Absolutely. We don't collect or store any personal data. All quiz content is processed securely and discarded immediately.

Can I use it on multiple devices?

Yes! Your Pro subscription works across all your Chrome browsers when logged in with the same account.
